## Contributing

clone the repository

```
git clone git@github.com:cbitstech/guides.git
```

install the Markdown lint tool

```
gem install mdl
```

run the Markdown lint tool as you work

```
mdl .
```

[make edits and submit a pull request](/git/pull_requests)
